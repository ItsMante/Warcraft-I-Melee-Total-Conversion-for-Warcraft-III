# Warcraft-I-Melee-Total-Conversion-for-Warcraft-III
Recreación completa del estilo jugable, unidades, economía y techtree de Warcraft: Orcs &amp; Humans, adaptado a los mapas de melee de Warcraft III. Incluye todos los stats, mejoras, estructuras y unidades originales, con ajustes modernos para funcionar en mapas de 8 o más jugadores.
